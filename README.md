# InteractableSpawnFix

Let you block interactable spawns from spawning in areas where they would be inaccessible or cause issues.

Press Insert to open the UI and start editing the spawn blocking areas.

You'll have to do the manual work of adding the areas, and it will save the areas to a file so you don't have to redo them every time.

In the future a default set of areas may be added to block spawns in the most common problem areas.
